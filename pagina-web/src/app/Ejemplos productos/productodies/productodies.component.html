Display LCD Gráfica 84 x 84 Nokia 5110

Descripcion.
El Nokia 5110 es una pantalla LCD gráfica básica para muchas aplicaciones. Fue pensada originalmente como una pantalla de teléfono celular. Éste se monta en un formato fácil de soldadura PCB. Se utiliza el controlador PCD8544, que es el mismo utilizado en el Nokia 3310 LCD. El PCD8544 es una de bajo consumo CMOS controlador LCD / conductor, diseñado para manejar una pantalla gráfica de 48 filas y 84 columnas. Todas las funciones necesarias para la visualización se proporcionan en un solo chip, incluyendo la generación de on-chip de suministro LCD y tensiones de polarización, lo que resulta en un mínimo de componentes externos y bajo consumo de energía. Las interfaces PCD8544 a los microcontroladores a través de una interfaz de bus serie.
